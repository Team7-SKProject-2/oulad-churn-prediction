# 프로젝트 요구사항

## 사용자

- 교육 서비스 운영 담당자
- 학습 상담 담당자
- 프로젝트 분석·모델링 담당자

## 문제 정의

개강 후 1~10주 동안 현재 주차까지 관측된 학생·과목 정보로 다음 주
중도이탈 확률을 예측한다. 위험 학생에게는 행동 신호와 과목 특성에
맞는 유지 활동을 제안한다.

## 분석 단위와 Target

- 분석 단위: 학생 × 과목 × 강좌 운영 회차 × 예측 주차
- Target: `target_next_week_withdrawn`
- 양성(1): 현재 수강 중이며 다음 주에 이탈
- 음성(0): 다음 주에 이탈하지 않음
- 출력: 다음 주 이탈확률, 위험 여부, 주요 위험 신호, 추천 유지 활동

## 조기경보 운영 범위

- 서비스 운영 구간: 1~10주차
- 최종 모델: CatBoost Enhanced 124 Feature
- 운영 임계값: `0.1100300614`
- 임계값 근거: 1~10주차 학생 단위 OOF 부분집합의 F1 최대
- 확률 보정: 별도 보정 미적용

최종 CatBoost는 전체 주차 학습 행으로 학습·OOF 검증했고, 1~10주차
Early 결과는 해당 OOF 행만 분리한 운영 구간 평가이다.

## 평가 지표

- 주지표: PR-AUC, F1
- 개입 지표: Recall, Precision, 경보 발생 비율
- 보조지표: ROC-AUC, Brier Score, ECE
- Accuracy만으로 모델을 선택하지 않는다.

## 완료 조건

- 동일 학생이 Fold 사이에 중복되지 않는다.
- 예측 주차 이후의 활동·평가·이탈 정보가 Feature에 포함되지 않는다.
- 1~10주차의 다음 주 이탈 성능을 주차·과목별로 확인할 수 있다.
- 위험 신호와 추천 유지 활동이 함께 출력된다.
- 최종 모델, 124개 Feature 순서, 임계값, 서비스 주차를 저장한다.
- Streamlit이 저장된 모델과 Early 운영 설정을 실제로 불러와 추론한다.
- README, EDA·모델 보고서, 발표 자료의 주제와 지표가 일치한다.
